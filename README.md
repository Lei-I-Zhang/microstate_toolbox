# +microstate toolbox
This repository contains the +microstate toolbox, a MATLAB toolbox for M/EEG sensor and source-space microstate analysis. 

Details of the methods underpinning the toolbox can be found in the following publications: 

[1] Tait and Zhang (2021) *MEG cortical microstates: spatiotemporal characteristics, dynamic functional connectivity and stimulus-evoked responses.* bioRxiv 2021.03.25.436979, doi: <a href="https://doi.org/10.1101/2021.03.25.436979">10.1101/2021.03.25.436979</a>

[2] Tait and Zhang (2021) *+microstate: A MATLAB toolbox for brain microstate analysisin sensor and cortical EEG/MEG*. Submitted to bioRxiv.
