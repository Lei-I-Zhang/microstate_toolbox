The files contained in this folder contain dipole positions used in the old
LORETA software and the newer sLORETA software. They are based on the file
fieldtrip-20200607/private/loreta_ind.mat which is used in the Fieldtrip
function loreta2fieldtrip to import LORETA/sLORETA data. These files are
used in the +microstate toolbox function individual.import_loreta.
